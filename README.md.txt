# Theme Toggle with Smart Tooltip - GSOC 

## Features 
- Tooltip perfectly aligned to theme icon
- Smart background colors (White in light mode, Dark in dark mode) 
- Keyboard accessible  
- Responsive design
- Production-ready code 

## How to Use 
1. Open 'index.html' in your browser 
2. Click the theme toggle button to switch between light/dark modes 
3. Hover over the button to see the smart tooltip
